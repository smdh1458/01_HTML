논리 연산자(&&, ||, ! )

AND 연산자(A && B)

        - A, B 모두 true일 경우 결과로 true를 반환하는 연산자
            (나머지의 경우는 모두 false)
            A = 조건1       B = 조건2
            A 와 B가 모두 참일 때만 true
            (내가 회원가입 할 때 작성한 값이 로그인을 시도했을 때)
            A = 아이디가 일치하는가?
            B = 비밀번호가 일치하는가?

로그인 하기

        - 회원가입 할 때 저장한 아이디 : 'member01'
        - 회원가입 할 때 저장한 비밀번호 : 'pass01'
        아이디와 비밀번호가 일치할 경우                 alert("로그인 성공~")

        아이디와 비밀번호가 하나라도 일치하지 않으면    alert ("아이디 또는 비밀번호가 일치하지 않습니다.")

OR 연산자(||)

      - A, B 둘 중 하나라도 true 일 경우 결과로 true 반환하는 연산자
        (A, B모두 false 일 경우에만 false)
      둘 중 하나만 참이어도 참으로 허용

      번호 
      회원가입 할 때 전화번호 나 휴대폰 번호 둘중 하나라도 입력

      A = 핸드폰 번호를 작성했는가
      B = 일반 전화번호를 작성했는가

전화번호 입력

논리 부정(NOT) 연산자(!)

      - 논리값을 반대로 바꾸는 연산자
      - 청개구리

      값이 false일 때 true

      예를 들어
      const 사과 = "apple";
            true
      사과 != "apple" ? "사과가 apple이 아니라 grape 입니까?" 
                      : "여기에 값이 보임"

부등화 (=, ==, ===)

      =   (할당 연산자)
        - 변수에 값(숫자나 문자)을 할당할 때 사용
        ex) const a = 5; let 사과 = "빨간색"; x = 5;

      ==  (동등 연산자)
        - 값만 비교하는 연산자. 타입이 다르더라도 값이 같으면 true 반환
        - ex)
              const 숫자5 = 5;
              const 글자5 = "5";
              숫자5 == 글자5가 같습니까? ====>  ok! 같네요.
                값을 우선적으로 보기 때문에 숫자나 글자가 형식의 값이 일치하면 같다고 표현
      
      === (일치 연산자)
        - 값과 타입이 모두 일치하는지 비교 값과 타일이 모두 같을 때만 true 반환
        - ex)
              const 숫자5 = 5;
              const 글자5 = "5";
              숫자5와 글자5가 같습니까?  ===> 하나는 문자고 하나는 숫자라서 다릅니다? false
              동등 연산자보다 엄격한 비교를 원할 때 사용
        ==  는 타입 변환이 자동으로 일어나 일어나는 기준을 알 수 없음
        === 는 타입 변환을 자동으로 일어나지 못하게 설정 후 비교
