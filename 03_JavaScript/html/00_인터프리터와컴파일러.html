컴파일러(compiler)와 인터프리터(Interpreter)/h1> 
        * 프로그래밍 언어로 작성한 코드를 컴퓨터가 이해할 수 있는 형태로 변환해 주는 프로그램
        컴파일러(compiler): 한국어로 모음을 구성하다.
                            코드 모음 코드 수집
                            사람이 사용하는 언어로 코딩을 진행한 전체 소스 코드를
                            한 번에 컴퓨터가 알아들을 수 있는 언어로 변환하고 변환이 완료된 파일을 생성하여 파일을 실행
                            사람이 만든 파일.java -> 사람이 만든 파일.class
                            실행할 때는 사람이 만든 파일.class 파일이 실행이 되기 때문에 빠름!
                            예시 언어: C, C++, C#, Java
    
    
        인터프리터(Interpreter): 한국어로 해석하다 
                                인터프리터는 소스 코드를 한 줄씩 해석하면서 바로 실행
                                코드를 작성한 다음 바로 실행 가능
                                한 줄씩 해석하면서 실행하기 때문에 컴파일러 방식에 비해 속도가 느림
                                예시언어: python, JavaScript, Ruby
