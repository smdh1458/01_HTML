영역 관련 태그

div 태그

        - 문서(document) 코딩(html) 내에서 구역이나 컨테이너를 지정하는데 사용
        
        - p태그와 다른 점 p태그는 글자만 구역
        
        - div 태그는 그림 동영상 등 여러가지를 태그 틀 안에 담을 수 있음
        - div 태그를 나가면 자동으로 줄바꿈(enter) 처리가 됨

span 태그

        - 이어서 작성 가능한 틀을 제공하는 태그

div와 span 비교하기

첫 번째 div
두번째 div
세번째 div
첫 번재 span 두 번재 span 세 번재 span

block 형식

<> - 화면(공간)을 수직 분할 (세로 화면을 가로로 잘라서 층을 나누는 행위) - 가로 한 줄 전체를 차지하기 때문에

inline 형식 태그

      - 화면을 수평 분할 (가로 화면을 세로로 잘라서 열을 나눔)
      - 가로에서 큰 공간을 차지하지 않고 사용하고 싶은 만큼만 공간을 차지한 다음에 옆자리를 내어줌
      - width,height 와 같은 크기 지정 속성 사용불가

      -inline 형식의 태그
      b(굵게), strong(굵게), i(기울이기), mark(글배경 색칠하기),span(영역 분할)

크기 지정 가능 여부 차이점

  • block: 크기 지정 가능
  • inline: 크기 지정 불가능

inline - block 형식

      - inline의 특징 중 수평 분할 (세로로 자름)
      - block의 특징 중 크기 지정 가능

      - 대표적인 태그 : img 태그, iframe 태그


      *iframe 태그
      - 웹 문서(html) 내부에 또 다른 웹 문서를 추가하는 태그
