        요소: <태그>내용물
        수직 방향에서 요소들을 어떻게 정렬할지 결정
        부모 컨테이너의 height(세로)가 지정되어 있을 때 효과가 직관적으로 보임

        align-items:
            flex-start: 요소들을 위쪽으로 정렬
            flex-end  : 요소들을 아래쪽으로 정렬
            center    : 요소들을 가운데로 정렬
            baseline  : 요소들을 텍스트 기준선이 동일한 선에 맞춰짐
            stretch   : 요소들이 컨테이너의 높이에 맞춰 늘어나고 정렬됨(기본값)
