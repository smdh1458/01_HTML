Transition 과 Transform

transition 속성

        transition : 요소의 속성이 변경될 때 그 변화가 즉각적으로 나타나는 대신, 
                     부드럽게 전환되도록 애니메이션 효과 추가
                transition: 속성명 전환시간 전환속도곡선 전환이 시작되기전 지연시간
                                                                대기했다가 변화하기
                    예를 들어 배경색을 빨간색으로 변경할 것임
                            3초후에 빨간색으로 변경 시작! 됐으면 좋겠고
                            변화는 0.5초 속도로 느리게 변환됐으면 좋겠음
                            일정한 속도로 변환했으면 좋겠음
                    transition: background-color 0.5s linear 3s;
                                속도와 몇 초 후에 시작하라는 표기는 자주 사용 X
                                (hover 나 active와 같은 효과에서 빨간색을 지정해야하는 것이지
                                    transition에서 색상자체를 지정하는 것은 아님)

transform 속성

        transform : 요소를 이동, 회전, 크기 조절, 뒤집기 등의 변환을 적용하는데 사용
            1. translate : 요소를 x, y 축 방향으로 이동
                transform: translate(10px, 20px); -> 가로 오른쪽으로 10px 세로 아래쪽으로 20px 이동
                                    숫자 +(플러스   3글자) 오른쪽 3글자 아래쪽 3글자
                                    숫자 -(마이너스 4글자) 왼쪽 2글자 + 위쪽 2글자
                
                    transform:  rotate(45deg);     -> 45도 회전 시계방향
                    transform: scale(1.5)          -> 태그와 태그안에 있는 값 = 요소의 크기를 1.5배 키움
                    transform: skew(10deg, 5deg) 가로 오른쪽으로 10도, 세로 오른쪽으로 5도 기울이기
                
                     예를 들어 
                     요소를 오른쪽으로 20px 아래로 10px 이동하고 크기를 1.2배로 키운다.
                     이것은 transform 기능을 2개 이상 사용할 때 작성하는 방법

                     transform: translate(20px, 10px) scale(1.2); 띄어쓰기로 구분하여 두가지 사용 가능
                                2개 이상 1개... 뭐든지 가능!

트랜지션과 트랜스폼

        트랜지션은 배경색 글자크기 글자색 가로세로 크기 등
        다양한 스타일 속성값에 속도 변화를 줄 수 있음

        트랜스폼은 이동하기 기울이기

        상대적으로 트랜지션이 트랜스폼보다 많이 사용
