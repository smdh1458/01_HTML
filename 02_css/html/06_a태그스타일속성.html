        하이퍼링크태그
        하이퍼링크태그

        a태그 내부에 속성으로 href 작성하면 글자색상과 밑줄이 생김
        글자 색상 기본적으로
        파란색 - 아직 클릭하지 않은 링크색상
        보라색 - 이미 클릭해서 방문한 링크색상

        a 태그는 사용자가 특정 행동을 했을 경우를 3가지 경우로 나누어 설정

        a:hover = 마우스를 링크 위에 올렸을 때 스타일을 설정
            주로 text-decoration : underline 밑줄 추가 설정

        a:active = 마우스를 클릭했을 때 활성화되는 스타일
            주로 글자색을 또 한번 변경하고 클릭시 약간 줄어들거나
            커지게 만들어줌

        a:visited = 사용자가 이미 방문한 링크의 스타일을 지정할 때 사용
            주로 color만 사용
        button input type = "submit" 과 a태그의 차이점
        
        button input type = "submit" 태그는 제출을 위한 용도로 주로 사용됨

        a href 태그는 특정 주소나 링크로 이동하기 위해 사용
